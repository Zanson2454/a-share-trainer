# A股投资训练系统

## 概述

面向个人投资者的 A 股学习训练系统。核心理念：**教练型助手**——不给买卖建议，而是引导用户形成自己的交易规则。

## 五维研究闭环

```
技术面 ←→ 基本面
  ↕         ↕
大盘环境 ← 热点新闻
  ↕
国家政策
```

## 架构

```
用户 (Hermes CLI / 钉钉)
  ↓
命令路由 (cli.py)
  ↓
┌──────────┬──────────┬──────────┬──────────┐
│ 学习模块  │ 选股模块  │ 分析模块  │ 回测模块  │
├──────────┴──────────┴──────────┴──────────┤
│           数据层 (AKShare)                 │
├──────────────────────────────────────────┤
│         Obsidian Vault 同步               │
└──────────────────────────────────────────┘
```

## 安全约束

- 不接券商账户
- 不自动下单
- 不用"必涨""稳赚""强烈买入"等表达
- 所有输出标注"仅用于学习研究"
